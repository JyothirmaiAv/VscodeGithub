# 🔐 Passwort-Checker

Ein Python-Programm zur Analyse der Passwortstärke.

Das Programm überprüft verschiedene Sicherheitskriterien und bewertet das Passwort als **Schwach**, **Mittel** oder **Stark**. Zusätzlich erhält der Benutzer Verbesserungsvorschläge für ein sichereres Passwort.
## 🚀 Features

✅ Überprüfung der Passwortlänge

✅ Erkennung von Großbuchstaben

✅ Erkennung von Kleinbuchstaben

✅ Erkennung von Zahlen

✅ Erkennung von Sonderzeichen

✅ Ausgabe von Verbesserungsvorschlägen

✅ Einstufung der Passwortstärke

## 📋 Bewertungskriterien

Das Passwort erhält Punkte für folgende Kriterien:

- Länge zwischen 8 und 12 Zeichen
- Länge von mindestens 12 Zeichen
- Mindestens ein Großbuchstabe
- Mindestens ein Kleinbuchstabe
- Mindestens eine Zahl
- Mindestens ein Sonderzeichen
